# 一个收集“奇怪学术期刊”的 PDF 仓库合集

这个项目用于收集整理一些比较“奇怪”的学术期刊论文 PDF。

有些期刊本身就带有幽默、讽刺或者实验性质，比如：
- 各种奇怪选题
- 看起来很正经但实际上很离谱的论文
- 偏娱乐性质的学术期刊

为了方便整理和下载，我使用 Python 编写脚本对这些期刊的论文 PDF 进行采集，并分别建立独立仓库存放。

本仓库只是 **索引仓库**，用于汇总所有期刊的 PDF 仓库。

## 项目结构

由于 PDF 文件体积较大，每个期刊单独建立一个仓库进行存储。

本仓库只负责：

- 汇总所有期刊
- 提供统一入口
- 后续新增期刊索引
